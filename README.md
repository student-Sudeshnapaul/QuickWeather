# 🌦️ QuickWeather App

A modern and responsive weather application that provides real-time weather updates for any city using the OpenWeather API.

🔗 **Live Demo:**
https://student-sudeshnapaul.github.io/QuickWeather/

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature (°C)
* 🤔 Feels-like temperature
* 💧 Humidity information
* 🌬️ Wind speed
* 👀 Visibility status
* 🎨 Dynamic background based on weather conditions
* ⚡ Fast and responsive UI
* 📱 Fully responsive design (mobile + desktop)

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Glassmorphism UI + Responsive Design)
* JavaScript (Vanilla JS)
* OpenWeather API

## ⚙️ How It Works

1. User enters a city name
2. App sends request to OpenWeather API
3. Weather data is fetched and displayed
4. Background changes based on weather condition

---

## 🔑 API Setup

This project uses OpenWeather API.

* Get your API key from: https://openweathermap.org/api
* Replace in `script.js`:

```javascript
const API_KEY = 'YOUR_API_KEY';
```
